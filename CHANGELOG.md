# Changelog

All notable changes to the **hosca-plugins** marketplace are documented here. This changelog tracks
the marketplace catalog itself (structure, plugins added/removed/renamed, marketplace-manifest
changes) — individual plugins keep their own changelog (e.g.
[`plugins/sql-audit-skill/CHANGELOG.md`](plugins/sql-audit-skill/CHANGELOG.md)).

Format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/); the marketplace manifest
`version` follows [Semantic Versioning](https://semver.org/). Release process:
[`RELEASING.md`](RELEASING.md).
